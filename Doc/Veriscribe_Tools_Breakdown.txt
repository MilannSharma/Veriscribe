SECTION C — TOOLS THAT NEED ONLY CODING & LIBRARIES (No AI Model Required)
================================================================================

These tools can be built fully using standard libraries, APIs, and algorithms.
No LLM or trained model is required.

──────────────────────────────────────────────────────────────────────────────
#   TOOL NAME                  LIBRARIES / STACK REQUIRED
──────────────────────────────────────────────────────────────────────────────
5   Plagiarism Check           Copyscape API / iThenticate API (web index access)
                               → TF-IDF + Jaccard similarity (string matching)
                               → Regex-based paraphrase detection
                               → Web scraper / search API integration

6   Citation Generator         CrossRef API (DOI metadata)
                               → Google Books API (ISBN metadata)
                               → Open Graph / meta tag parser (URL metadata)
                               → CSL (Citation Style Language) JSON processor
                               → citation-js npm library

7   Doc Editor                 ProseMirror / TipTap (rich text editor)
                               → Mammoth.js (DOCX parsing)
                               → PDF.js + pdf-lib (PDF read/write)
                               → SheetJS (XLSX parsing)
                               → LibreOffice headless (server-side conversion)
                               → diff-match-patch (tracked changes)

16  File Converter             LibreOffice headless (Office format conversions)
                               → Ghostscript (PDF operations: merge, split, compress)
                               → ImageMagick (image conversions)
                               → Tesseract OCR (Scan to PDF, OCR PDF)
                               → FFmpeg (MP4 to MP3, Video to Text)
                               → Whisper (Video to Text — speech only, not LLM)
                               → qpdf / pdftk (PDF manipulation)
                               → Sharp (image processing — JPG, PNG, WEBP)
                               → pdf2pic (PDF to image)

18  Document Version           diff-match-patch (line-by-line diff engine)
    Reconciler                 → jsondiffpatch (structured diff for JSON/DOCX XML)
                               → Mammoth.js (DOCX parsing)
                               → Rule-based clause risk flagging (regex + keyword)

19  Document Archaeology       Google Drive API / Notion API (version history access)
                               → git-style version graph (DAG data structure)
                               → Comment thread aggregator
                               → File metadata parser

20  Document Resurrector       File metadata scanner (last modified, word count)
                               → Completeness heuristics (section detection, word count)
                               → Template matching engine (structure inference)
                               NOTE: The "AI completion draft" feature DOES need an LLM.
                                     Completeness scoring + draft detection = code only.

21  Staleness Monitor          Google Drive API / Notion API (watch for changes)
                               → Webhook / polling engine
                               → Link extractor (finds referenced sources in docs)
                               → Diff checker (detects source changes)
                               → Email / push notification service

22  Document Analytics         Custom event tracking SDK (like Segment / Mixpanel)
                               → Read-event pixel (tracks opens, section time)
                               → Scroll-depth tracker (JS IntersectionObserver)
                               → Heat map renderer (Canvas API or heatmap.js)
                               → Time-on-section timer (requestAnimationFrame)

23  Confidence Scorer          Rule-based citation extraction (regex + NLP patterns)
    & Trust Shield             → YARA rules / malware signature scanner (file scan)
                               → VirusTotal API (malicious content check)
                               → Trust score calculator (weighted rule engine)
